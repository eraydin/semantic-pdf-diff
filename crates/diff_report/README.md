# diff_report

Stable JSON, AI review JSON, Markdown, and self-contained HTML report
generation with deterministic inline SVG evidence overlays.

Markdown change lists include available old/new node evidence text and page
references so generated HTML carries the same semantic content.
HTML diff reports render old/new evidence side by side and include available
page and bounding-box evidence in PDF user space. When changed nodes include
bounding boxes, HTML reports also include inline SVG page evidence overlays
without external network resources.
